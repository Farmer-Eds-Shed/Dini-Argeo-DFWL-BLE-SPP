# Dini-Argeo-DFWL-BLE-SPP
ESP32 Sketches to Connect TTL Weigh head to BLE App (HerdApp) and SPP Wand (Agrident AWR)


It extracts `21.0` and sends:
- BLE payload matching Tru-Test S3 spec
- SPP message: `21.0 kg`
